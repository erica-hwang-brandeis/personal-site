Part 1: Paintbrush Holder on Tinkercad

Tinkercad

Remixing using Tinkercad to create my own version of a design!

button

Design

Pain point trying to solve: When I paint, my desk tends to become so messy that there is no space to put the things I need on top. So I want to try to save space by having a place for my paint brushes in between each use, specifically with vertical stacking to minimize the space that the paint palette and paint brushes take up.

Paintbrush Holder

I found a simple paintbrush holder design on thingiverse, and put the stl file into tinkercad to remix the design. I used cylinders to make holes in the four corners that would allow cylinder support beams to be inserted in. The idea is that then I could stack the paint palette on top of the brush holder. I also used the text feature to add the name of my art brand inside the box.

Download STL

Slicing and Initial Print

When slicing the file, I added 2 mm of brims to make the small cylinder supports stick better since they are very skinny. The start of the print went smoothly. However, when I came back to the print a couple days later, the print looked a bit messed up. There was warping at the base of the box, with the four corners curling upwards. The two layer shifts caused the separate pieces to run into each other later in the print, resulting in a goopy slug. The base was not big enough to support the columns since they were almost tipped over as it finished printing, so I would need to increase that setting for the cylinders to stand properly.

Fixing

With help from Sam Shmidman, I re-printed the design. Before printing, we redesigned the brush holders to have the columns built-in instead of being attachable, which I learned is difficult to do with this type of circular fit. I also added brims to keep the edges from curling upwards again. The second print came out successful, and it was much sturdier and cleaner.

Second iteration results

I ended up getting lucky because I was just guessing the dimensions without having my palette next to me. Also, since it is a circular palette, I wasn’t really sure how to optimize the columns to fit well. However, it ended up fitting perfectly and also nice and snug because of the angled edge of the circle. The concavity worked well for holding brushes of various sizes. In a future iteration, I would lower the height of the base while keeping the columns the same size. The text didn’t come out very clear, so maybe I would make the text bigger or print at a slower speed. I haven’t tried using it while painting yet, but I will share updates once I do about its functionality and ergonomicness!

Part 2: QR Code Keychain on Fusion

Fusion

Designing in Fusion for the first time!

button

Design

I have 3-d printed a keychain with my Stretch Stop logo before, but I wanted to try another iteration that has a more fun shape. Also, I wanted it to have a qr code on it so that I can easily direct people to the instagram page. Previously, I printed the qr code on a small label maker and stuck it to the back of the keychain, but it didn’t work very well. So I decided to try 3-d printing the qr code directly on the keychain.

Download STL

Slicing and Printing

I found a stick figure on thingiverse that looked like it is stretching and remixed the design to add the qr code and “Stretch Stop”. I used fusion to fuse together the qr code onto the stick figure. I did this by converting the png of the qr code to a svg file, then imported into fusion, and extruded it by 4 mm. I did the same process for the “Stretch Stop” letters.

Slicing and Printing

I realized that I didn’t know how to set it up so that the prusa mini knows when to stop to change filament color. Greg and Abby at the MakerLab helped me out. We ended up recreating the files separately on fusion, and adding them together in Prusa Slicer so that we could indicate the height of the color change.

Starting over the print

When printing, the first layer looked fine but then the second layer started coming off of the bed. It was dusty so the print didn’t stick, but after wiping it down and starting over, that stick figure part printed with no issues. I changed the filament color to a dark blue for the qr code and letters so that there is contrast. However, I realized that the qr code was and letters were too small, so the parts started stringing together a little, so the qr code did not end up working.

Fixing

During office hours at the engineering classroom, we tried testing to see what is the smallest size the Prusa Mini can print a functional qr code. The qr code on the original stick figure print was (check) a 1.5 mm wide square. We printed the qr code on its own, but made it 3 mm.

Design

This print turned out to be successful, with the qr code working. I haven’t had the chance to make new iterations of it, but I plan to keep testing how much smaller I can make the qr code so that it would fit well on the keychain.
